# Дашборд эффективности команды разметки

**Интерактивный дашборд:** [открыть в DataLens](https://datalens.yandex/1hv0y2j7azj6l)

## Задача проекта

Оценить эффективность трёх команд разметки (Альфа, Бета, Гамма) по объёму, скорости, качеству и дисциплине. Выявить точки роста и дать рекомендации.

## Данные

- Период: май 2026
- Метрики: количество размещённых пабликов, скорость (пабликов/час), количество ошибок, нарушения дисциплины, цвет карточек (зелёный — отлично, белый — средний)

## Ключевые выводы

- Общий объём: **29 160 пабликов**
- Лидер по объёму: **Гамма** (9 910 пабликов)
- Лучшая точность: **Альфа и Гамма** (0,81% и 0,78% ошибок)
- Зона внимания: **команда Бета** (0,98% ошибок, выше среднего)
- Корреляция: с ростом скорости количество ошибок в среднем растёт, но есть исключения (опытные сотрудники)

## Рекомендации

1. Провести тренинг для команды Бета по снижению ошибок
2. Ввести план по зелёным карточкам (минимум 3 на команду)
3. Добавить в дашборд динамику по месяцам для отслеживания прогресса

## Как использовать дашборд

Дашборд интерактивный. Вы можете наводить на графики, смотреть значения, переключать команды (если добавлены фильтры).

## Использованные инструменты
